01_제이쿼리 문서객체모델_탭메뉴 콘텐츠

노랑풍선 사이트의 '해외여행패키지 > 괌' 페이지의 탭 메뉴 콘텐츠 구현

탭 메뉴 콘텐츠는 메뉴 클릭 시 해당하는 콘텐츠가 바로 아래 나오면서 기존 콘텐츠는 숨겨지고, 영역 안에서만 변경되는 콘텐츠이다

특정 영역에서만 보여지기 때문에 좁은 공간에서 많은 콘텐츠를 보여 주고자 할 때 사용

구현 방법
1. html 목록 태그를 사용하여 목록(li)에 a 요소 4개와 div 요소를 사용하여 콘텐츠 영역 4개를 만들고 css를 활용하여 첫 번째 콘텐츠만 보이도록 스타일 서식 지정
2. tab 메뉴(a 요소)를 클릭하면 해당 콘텐츠(div)가 보이도록 하고 나머지 콘텐츠들은 숨긴다 이때 addClass() 메소드를 사용한다
3. 메뉴 서식은 선택한 메뉴에만 적용되도록 하고 그 외 나머지 메뉴는 적용된 서식을 제거한다 (removeClass)

사용할 메소드: addClass(), removeClass(), siblings(), find(), css()
