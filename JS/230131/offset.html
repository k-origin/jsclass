02jquery_속성 관련(offset, 위치)

1. offset 메소드

사용자가 선택한 요소의 위쪽이나 왼쪽을 기준으로 브라우저에서 떨어져 있는 위치 값을 구하고자 할 때 사용하는 메소드이다

문법) $('대상').offset(), $('대상').offset().top, $('대상').offset().left

2. scroll 메소드

scroll 메소드는 윈도우의 스크롤바의 위치 값을 반환하고 설정해 주는 메소드이다(가로, 세로)

  • scrollTop(); - 스크롤바의 수직 위치 값 설정
  • scrollLeft(); - 스크롤바의 가로 위치 값 설정
세로 스크롤 값 나오는 곳 가로 스크롤 값 나오는 곳 Top
