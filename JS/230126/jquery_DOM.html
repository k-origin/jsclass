jQuery_DOM_class 조작

문서객체모델(DOM) - class 조작은 사용자가 선택한 요소에 클래스 서식을 추가하거나 삭제 또는 클래스가 없는 요소를 찾아서 서식을 적용하는 메소드이다

  • addClass(); - 선택한 요소에 클래스 서식을 추가한다
  • removeClass(); - 선택한 요소에 클래스 서식을 제거한다
  • toggleClass(); - 선택한 요소에 클래스 서식을 추가하거나 제거한다
  • hasClass(); - 선택한 요소에 해당 클래스 이름이 있는지 여부를 판단하여 true, false 값으로 표시
  • css(); - 사용자가 선택한 요소에 스타일 서식을 적용한다
