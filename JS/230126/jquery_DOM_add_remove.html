제이쿼리 문서객체모델(DOM) - 내용 추가, 삭제

객체 편집 메소드는 html 문서 안에서 사용자가 선택한 요소의 앞쪽이나 뒤쪽에 내용을 추가하거나 복제, 삭제하는 메소드

1. 내용 추가하기(add)

  • append(); - 선택한 요소(부모)의 안쪽으로 맨뒤에 내용을 추가하기
  • prepend(); - 선택한 요소(부모)의 안쪽으로 맨앞에 내용을 추가하기
  • after(); - 선택한 요소(형제)의 뒤에 내용을 추가하기
  • before(); - 선택한 요소(형제)의 앞에 내용을 추가하기
  • insertBefor(); insertAfter(); - 자리 배치하기

내용 추가 메소드는 이메일 추가, 쇼핑몰 콘텐츠 '더보기', 모달 윈도, 갤러리 작품 더보기, 공지사항 더보기, FAQ 더보기 등 내용을 추가하고자 할 때 응용할 수 있다

insertAfter(), insertBefore()를 사용하면 A라는 요소의 앞쪽에 B라는 요소를 배치할 수 있고, B라는 요소의 뒤쪽에 A 요소를 배치할 수도 있다. after(), befoer() 메소드와 기능은 같지만 사용법이 다르다

    예)
    $('A').after('B'); - A 요소의 바깥쪽으로 뒤에 B 내용을 추가한다 AB 

    $('B').insertAfter('A'); - A 요소의 바깥쪽으로 뒤에 B 내용을 추가한다 AB

내용 삭제하기(Remove)

사용자가 선택한 요소를 찾아서 내용만 지우거나, 태그 요소 자체를 삭제할 수 있다

  • empty() : 사용자가 선택한 요소의 내용만 삭제
  • remove() : 사용자가 선택한 요소를 삭제
